## Build a drag and drop file manager | Frontend Interview Question [Hindi]

This repository contains the source code for the YouTube video [Build a drag and drop file manager | Frontend Interview Question [Hindi]](https://youtu.be/LPBAz3Vj2uw).

In this video tutorial, we'll guide you through building a drag-and-drop file manager using vanilla JavaScript. Learn how to display and interact with a hierarchical file system where folders can contain both files and other folders. We'll implement drag-and-drop functionality to move files and folders, ensuring proper sorting and organisational rules.
